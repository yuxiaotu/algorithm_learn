# 单链表

- 在数组中，要插入或者是删除一个元素，需要移动元素。
- 而链表可以弥补这个缺陷，链表在内存中存放的位置不是连续的，是通过 `next` 属性指向下一个元素。
- 当需要插入或者是删除时，只要改变 `next` 的指向即可。


## 01. JavaScript
```js
class Node {
  constructor(value) {
    this.value = value;
    this.next = null;
  }
}
```
```js
class LinkedList {
  constructor() {
    this.head = null;
    this.tail = this.head;
    this.length;
  }

  append(value) {
    const newNode = new Node(value);
    if (!this.head) {
      this.head = newNode;
      this.tail = newNode;
    } else {
      this.tail.next = newNode;
      this.tail = newNode;
    }
    this.length++;
  }

  prepend(value) {
    const node = new Node(value);
    node.next = this.head;
    this.head = node;
    this.length++;
  }

  insert(value, index) {
    if (index >= this.length) {
      this.append(value);
    }
    const node = new Node(value);
    const {prevNode, nextNode} = this.getPrevNextNode(index);
    preNode.next = node;
    node.next = nextNode;
  }

  getPrevNextNodes(index) {
    let count = 0;
    let prevNode = this.head;
    let nextNode = prevNode.next;
    while (count < index - 1) {
      prevNode = prevNode.next;
      nextNode = prevNode.next;
      count++;
    }
    return {prevNode, nextNode};
  }

  remove(index) {
    let {prevNode, nextNode} = this.getPrevNextNodes(index);
    prevNode.next = nextNode;
    this.length--;
  }
}
```
