# [对称二叉树](https://leetcode-cn.com/problems/symmetric-tree/)

- 使用队列，将互为镜像的结点入队。
- 当两个子树的根结点相等时，比较左子树和 `left` 和 右子树的 `right`。
- 将左子树的 `left` 和右子树的 `right` 入列。
- 然后将左子树的 `right` 和右子树的 `left` 入列。

## 01. JavaScript

}
