# 二叉搜索树实现

## 01. 含义
[二叉树搜索树](https://zh.wikipedia.org/wiki/%E4%BA%8C%E5%85%83%E6%90%9C%E5%B0%8B%E6%A8%B9) 满足二叉树的性质，同时对于任意结点，其左子树中的关键字不大于父结点的关键字，而其右子树的关键字不小于父结点的关键字。对二叉树的操作主要有遍历、查找、插入、删除等。



## 02. 二叉树的遍历模式
按照访问根结点的顺序可以将二叉树的遍历分为以下三种情况：
- #### 先序遍历
  先访问根结点，然后访问左子树，最后右子树。
- #### 中序遍历
  先访问左子树，然后访问根结点，最后访问右子树。
- #### 后序遍历
  先访问左子树，然后访问右子树，最后访问根结点。



## 03. 递归法遍历的伪代码
先序遍历：
```c++
preorder_tree_walk(root) {
  if (root != null) {
    print root.val
    preorder_tree_walk(root.left)
    preorder_tree_walk(root.right)
  }
}
```

中序遍历：
```c++
inorder_tree_walk(root) {
  if (x != null) {
    inorder_tree_walk(root.left)
    print root.val
    inorder_tree_walk(root.right)
  }
}
```

后序遍历：
```c++
postorder_tree_walk(root) {
  if (root != null) {
    postorder_tree_walk(root.left)
    postorder_tree_walk(root.right)
    print root.val
  }
}
```



## 04. 迭代法遍历的伪代码
借助于「栈」的结构，通过「栈」的后进先出来模拟递归中函数调用与函数执行返回的过程。

[先序遍历](https://leetcode-cn.com/problems/binary-tree-preorder-traversal/solution/er-cha-shu-de-qian-xu-bian-li-by-leetcode-solution/)：

- 先将根结点入栈。
- 当栈不为空时依次弹出栈顶元素。
- 访问弹出的栈顶元素。
- 然后分别将弹出的栈顶元素的左子树和右子树入栈。
- 循环往复直到栈为空。

```c++
preorder_tree_walk(root) {
  stack stk
  stk.push(root)
  while (node != null || !stk.empty) {
    node = stk.pop()
    print node.val
    if (node.right != null) {
      stk.push(node.right)
    }
    if (node.left != null) {
      stk.push(node.left)
    }
  } 
}
```

[中序遍历](https://leetcode-cn.com/problems/binary-tree-inorder-traversal/solution/er-cha-shu-de-zhong-xu-bian-li-by-leetcode-solutio/)：

- 首先将所有左子树的根结点入栈。
- 依次弹出栈顶元素。
- 访问弹出的栈顶元素。
- 将弹出的栈顶元素的右子树入栈。
- 循环往复直到栈为空。

```c++
inorder_tree_walk(root) {
  stack stk
  while (root != null || !stk.empty) {
    while (root != null) {
      stk.push(root)
      root = root.left
    }
    node = stk.pop()
    print node.val
    root = node.right
  }
}
```

[后序遍历](https://leetcode-cn.com/problems/binary-tree-postorder-traversal/solution/er-cha-shu-de-hou-xu-bian-li-by-leetcode-solution/)：

- 首先将所有左子树的根结点入栈。
- 依次弹出栈顶元素。
- 将弹出的栈顶元素的右子树入栈。
- 访问弹出的栈顶元素。
- 循环往复直到栈为空。

```c++
postorder_tree_walk(x) {
  stack stk
  stk.push(root)
  while (root != null || !stk.empty) {
    while (root != null) {
      stk.push(root)
      root = root.left
    }
    root = stk.pop()
    root = root.right
    print root.val
  }
}
```



## 03. 查找二叉搜索树中的关键字
输入一个指向树根的指针和一个关键字 `k`, 如果这个 `k` 存在则返回指向这个结点的指针。

根据二叉搜索树的性质，当 `k` 小于根结点的值时，就在左子树中查询；当 `k` 大于根结点的值时，就在右子树中查询。

递归法查找：
```c++
tree_search(root, k) {
  if (root == null || k == root.key) {
    return root
  } 

  if (k < root.key) {
    return tree_search(root.left, k)
  } else {
    return tree_search(root.right, k)
  }
}
```

迭代法查找：
```c++
iterative_tree_search(root, k) {
  while (root != null && root != x.key) {
    if (k < x.key) {
      root = root.left
    } else {
      root = root.right
    }
  }
}
```



## 03. 查找最大值或最小值
根据二叉搜索树的性质，从根开始沿着左子树直到遇到一个 `null`，总能找到一个结点是最小值。

同样沿着右子树直到遇到一个 `null`，总能找到一个元素是最大值。

```c++
tree_minimum(root) {
  while (root.left != null) {
    root = root.left
  }
  return root;
}
```

```c++
tree_maximum(root) {
  while (root.right != null) {
    root = root.right
  }
  return root
}
```



## 04. 查找后继或前驱
二叉搜索树可以通过没有任何关键字的比较来确定一个结点的后继。当所有结点互不相同时，一个结点 `node` 的后继是大于 `node.key` 的最小值的结点。

```c++
tree_successor(node) {
  if (node.right != null) {
    return tree_minimum(node.right)
  }

  temp = node.p
  while (temp != null && node == temp.right) {
    node = temp
    temp =temp.p
  }
  return temp
}
```


## 05. 插入和删除
将一个新值 `v` 插入到一颗二叉搜索树 `T` 中，以结点 `z` 作为输入结点，其中 `z.key = v`，`z.left = null`，`z.right = null`。

指针 `x` 记录了一条向下的简单路径，遍历指针 `y` 作为 `x` 的双亲，

`while` 循环使得这两个指针沿着树向下移动，向左或者是向右移动取决于 `z.key` 和 `y.key` 的比较。

当 `x` 变为 `null`，这个 `null`k 就占据的位置就是 z 要放置的地方。

```c++
tree_insert(T, z) {
  y = null
  x = T.root
  while (x != null) {
    y = x
    if (z.key < x.key) {
      x = x.left
    } else {
      x = x.right
    }
  }
  z.p = y
  if (y == null) {
    T.root = z
  } else if (z.key < y.key) {
    y.left = z
  } else {
    y.right = z
  }
}
```
