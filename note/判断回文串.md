# [判断回文串](https://leetcode-cn.com/problems/valid-palindrome/)

- 用正则表达式去掉无关字符。
- 设置 `R` 和 `L` 指针，指向字符串首部和尾部。
- 指针 `R` 和 `L` 同时向中间移动。

## 01. JavaScript
```js
var isPalindrome = function(s) {
  s = s.replace(/[\W|_]/g, " ").toLowerCase();
  if (s.length < 2) {
    return true;
  }
  let L = 0;
  let R = s.length - 1;
  while (L < R) {
    if (s[L] !== s[R]) {
      return false;
    }
    L++;
    R--;
  }
  return true;
}
```
