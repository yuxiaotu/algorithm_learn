# [跳跃游戏](https://leetcode-cn.com/problems/jump-game/)

##  01. JavaScript
```js
public boolean canJump(nums) {
  let len = nums.length;
  let step = 0;
  for (let i = 0; i < len; i++) {
    if (i <= step) {
      step = Math.max(step, i + nums[i]);
      if (step >= len - 1) return true;
    }
  }
  return false;
}
```
