# [反转字符串](https://leetcode-cn.com/problems/reverse-string/)

## 01.

```java
// java
class Lc344 {
  public void reverseString(char[] s) {
    if (s.length == 0 || s.length == 1) return;
    int L = 0;
    int R = s.length - 1;
    while (L < R) {
      char temp = s[L];
      s[L++] = s[R];
      s[R--] = temp;
    }
    return;
  }
}
```
