# [杨辉三角](https://leetcode-cn.com/problems/pascals-triangle/)


## 01. JavaScript
```js
var generate = function(numRows) {
  let dp = [[1]];
  for (let i = 1; i < numRows; i++) {
    dp[i] = [];
    for (let j = 0; j < i; j++) {
      dp[i][j] = (dp[i-1][j] ? dp[i-1][j] : 0) 
               + (dp[i-1][j-1] ? dp[i-1][j-1] : 0);
    }
  }
  return dp;
}
```
