# [删除重复项](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array/)

- 定义 `fast` 和 `slow` 两个指针。
- 同时指向 `nums[1]`，也就是从第二个元素开始比较。
- `fast` 指针先向后运动，

## 01. JavaScript
```javascript
var removeDuplicates = function(nums) {
  const len = nums.length;
  if (len == 0) {
    return 0;
  }
  let fast = 1;
  let slow = 1;
  while (fast < len) {
    if (nums[fast] != nums[fast-1]) {
      nums[slow] = nums[fast];
      ++slow;
    }
    ++fast;
  }
  return flow;
}
```
